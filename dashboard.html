Web Programing

Pemrograman web adalah proses pengembangan aplikasi yang beroperasi di web browser atau server untuk membangun dan mengelola interaksi antara pengguna dan website. Pemrograman web merupakan cabang ilmu dari pemrograman komputer yang mengimplementasikan konsep pemrograman di dunia web. Pemrograman web erat kaitannya dengan internet dan website, karena merupakan proses pembuatan website untuk keperluan internet.

Mobile Programing

Pemrograman mobile adalah proses pembuatan aplikasi atau perangkat lunak yang dirancang untuk berjalan pada perangkat mobile, seperti ponsel, tablet, dan asisten digital. Dalam pemrograman mobile, programmer menggunakan bahasa pemrograman yang dirancang untuk perangkat mobile. Salah satu bahasa pemrograman yang populer untuk pengembangan aplikasi Android adalah Java.
